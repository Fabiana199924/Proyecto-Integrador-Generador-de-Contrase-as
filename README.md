# Proyecto Integrador: Generador de Contraseñas Seguras

## Integrantes:
* Fabiana Nicolle Montenegro Brito

## Objetivo del Programa:
Ayudar a las personas a proteger sus cuentas en internet creando contraseñas de forma aleatoria y evaluando si son seguras según su tamaño.

## Funcionalidades Principales:
1. Pide al usuario la longitud de la contraseña.
2. Usa un bucle `while` para juntar caracteres al azar (letras, números y símbolos).
3. Clasifica la contraseña en Débil, Regular o Fuerte usando condiciones `if`.
