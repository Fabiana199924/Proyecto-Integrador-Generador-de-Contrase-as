# Proyecto Integrador: Generador de Contraseñas Seguras

## Integrantes:
* Fabiana Nicolle Montenegro Brito

## Objetivo del Programa:
Ayudar a las personas a proteger sus cuentas en internet creando contraseñas de forma aleatoria y evaluando si son seguras según su tamaño.

## Funcionalidades Principales:
1. **Entrada de datos:** Pide al usuario la longitud que desea para su contraseña.
2. **Estructura Repetitiva (`while`):** Junta caracteres al azar (letras, números y símbolos) uno por uno hasta alcanzar el tamaño solicitado.
3. **Estructura Condicional (`if`):** Evalúa si la contraseña es menor a 8 caracteres para marcarla como DÉBIL.
4. **Estructura Condicional (`elif`):** Evalúa si el tamaño está entre 8 y 11 caracteres para clasificarla como REGULAR.
5. **Estructura Condicional (`else`):** Determina que si tiene 12 o más caracteres, la contraseña es FUERTE.
